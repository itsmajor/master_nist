
## EMBLEM

EMBLEM and R.EMBLEM is a post-quantum cryptographic library based on learning with errors (LWE) problem written in C.


## Contents

This folder contains KAT, Reference Impelemtation, Optimized implementation and supporting documentation


## Building

All source codes uses OpenSSl's libcrypto, thus OpenSSL must be installed before building EMBLEM and R.EMBLEM

## To complie on Ubuntu:
just type

make

## To compile KAT on Ubuntu:

just type

make

## Parameters

Both EMBLEM and R.EMBLEM parameters can be configured by editing 'params.h'
